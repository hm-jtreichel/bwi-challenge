# bwi-challenge
Coding-Challenge der BWI

Genutzte Python-Version: 3.8.2

Anleitung:
Repository klonen bzw. 'cargo.py','main.py','items_list.csv' herunterladen und am gleichen Ort/Ordner speichern ('cargo_list.csv' ist optional - wird beim Ausführen erzeugt)

'items_list.csv' ist der Input für den Algorithmus; die Datei ist bereits nach Angaben in der Aufgabenstellung befüllt (Format: ID;ANZAHL;GEWICHT;NUTZWERT;NAME).

Python-Script 'main.py' in CMD/Terminal (innerhalb des Ortes/Ordners) ausführen: 'python main.py'

Erstellt - oder überschrieben, falls vorhanden - wird eine 'cargo_list.csv'-Datei, in der Name und Anzahl der jeweiligen Geräte aufgelistet sind (nach Transportern getrennt).
